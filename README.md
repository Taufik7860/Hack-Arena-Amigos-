Project:

Problem: Offices and industrial factories waste significant amounts of electricity by cooling empty rooms. Traditional ceiling fans and HVAC systems often run at full speed regardless of actual occupancy or ambient temperature, leading to high utility costs and a larger carbon footprint.

Solution: ClimateSense IONET is an AI-powered mesh network of occupancy and temperature sensors that intelligently regulates office ceiling fans.

Smart Optimization: Uses a "Digital Twin" floorplan to monitor real-time occupancy and adjust fan speeds automatically.

Financial ROI: Provides a live dashboard calculating financial savings (at ₹8.5/kWh) and CO2 reduction.

ESG Automation: Features a built-in reporting engine that automatically generates and emails monthly sustainability compliance reports to management.

Scalable Infrastructure: Built with a MERN stack and Socket.io to handle real-time telemetry across a distributed office network.
